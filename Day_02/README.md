# Day 2: Tip Calculator 💸

## Project Overview
A utility script designed to calculate the total bill per person, including a custom tip percentage. This project focuses on handling user input and performing mathematical operations with different data types.

## Key Learning Objectives
* **Data Type Manipulation:** Converting strings from `input()` into `integers` and `floats` for calculation.
* **Mathematical Operations:** Using the standard order of operations (PEMDAS) to calculate percentages.
* **Type Casting:** Resolving `TypeErrors` by explicitly changing data types.
* **Number Formatting:** Using **F-Strings** and the `round()` function (or format specifiers) to display currency correctly with two decimal places.

## How it Works
1. The user enters the total bill amount.
2. The user selects a tip percentage (10, 12, or 15).
3. The user specifies how many people are splitting the bill.
4. The script calculates the final share for each person.
