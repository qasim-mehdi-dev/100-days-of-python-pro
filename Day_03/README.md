# Day 3: Treasure Island Adventure 🏝️

## Project Overview
A "Choose Your Own Adventure" text-based game that utilizes complex conditional logic to navigate the user through a dangerous island. The goal is to reach the treasure by making the correct logical choices.

## Key Learning Objectives
* **Control Flow:** Mastering `if`, `elif`, and `else` statements to direct program execution.
* **Nested Logic:** Creating decisions within decisions to handle multi-layered scenarios.
* **Comparison Operators:** Using `==`, `>`, `<`, `!=` to evaluate user input.
* **Logical Operators:** Combining conditions using `and` and `or`.
* **Case Sensitivity:** Normalizing user input (e.g., `.lower()`) to ensure the game doesn't crash on "Left" vs "left".

## How to Play
1. Run the script in the terminal.
2. Follow the prompts to move Left/Right, Swim/Wait, or choose a Door.
3. Choose wisely—one wrong move ends the game!
